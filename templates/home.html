About LogFlow

LogFlow is a suite of tools for security professionals to process, analyze, and manage security event logs. The platform offers:

  • Liquid Template Processing - Transform JSON logs into clean, consistent outputs using Liquid.
  • Log Analysis - Analyse log files (JSON format) and pick key variables apart.
  • Sample Log Building - Create and manage sample logs for testing and training Liquid templates.
Getting Started
  1. Upload a JSON log file
  2. Create or upload a Liquid template
  3. Process your logs to generate reports
  4. Save your templates for future use
